OS X 10.11中Xcode找不到模拟器的解决方法

发表于 2015-07-27 | 分类于 iOS

yozoo原创稿件，转载注明出处
关于Xcode在OS X EI Capitan Beta 4 (15A226f)和OS X EI Capitan Beta 5(15A235d)环境下运行出现模拟器找不到或安装不到的问题

在升级完OS X EI Capitan Beta 4之后打开Xcode时会发现模拟器不见了，不能运行程序，连接手机后也不可以。无论是Xcode6.4和Xcode 7 Beta，都是一样，在App Store重新下载也是这样。

之后在苹果开发者论坛(https://forums.developer.apple.com/message/33128#33128)看到了一些解决方法

打开“终端”
1.到runtimes文件夹 

cd /Library/Developer/CoreSimulator/Profiles/Runtimes

2.删除文件夹里面的内容

rm -rf *

如果需要管理员权限的，输入

sudo rm -rf *

重新打开xcode就可以了
